## Projects

### 📊 [DotnetMappingBenchmarks](https://github.com/jagobainda/DotnetMappingBenchmarks)
Comparative benchmarks of major object mapping libraries for .NET, executed with BenchmarkDotNet on .NET 10. Results persist as static JSON files and are served to the companion website.
